Casus FriendlyTournament

Het idee is het maken van een tournament applicatie waar je als user in een groep kan zitten die samen in kunnen schrijven in een tournament. Verder kan je vrienden hebben en bv. zien of hij/zij al in een groep zitten. Het erd is hieronder te vinden.

User Stories:

Scenario1:
Een User logt succesvol in en maakt een nieuwe groep aan en sluit zich automatisch aan bij de gemaakte groep. De user nodigt zijn/haar vriend uit zich aan te sluiten aan de groep.

Scenario2:
Een bestaande user krijgt een uitnodiging naar een groep en accepteert die. De user is aangesloten bij die groep.

Scenario3:
Een groep met genoeg spelers schrijft zich op tijd (en dus succesvol) in voor een tournament.

Deze applicatie is gemaakt in het kader van het vak 'Clientside web frameworks' van Avans HogeSchool Breda.
